# Portfolio Cases

Este repositório reúne estudos de caso de projetos reais em que participei como desenvolvedor.

Os códigos-fonte originais não estão públicos por motivos comerciais, legais e estratégicos. Cada pasta contém contexto, responsabilidades, stack, screenshots e links públicos quando disponíveis.
